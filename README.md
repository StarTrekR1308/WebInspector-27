# 📱 WebInspector 27 (iPadOS Touch Mode)

Ein haptischer, touch-optimierter F12-Entwickler-Editor für Safari auf dem iPad – gebaut im modernen Design-System. Schmeiß die unbedienbaren Desktop-Overlays über Bord und editiere das Web mit echten iOS-Buttons.

## ✨ Features
* 🟩 **Text bearbeiten:** Elemente live antippen und Texte auf Webseiten umschreiben.
* 🟦 **Design anpassen:** CSS-Styles (Farben, Radien, Schatten) per Touch-Popup injizieren.
* 🟥 **Element löschen:** Kaputte oder störende Web-Elemente mit einem Tap restlos entfernen.
* 🎛️ **Clean Screenshot:** Blendet das Interface kurz aus, um fehlerfreie Screenshots zu knipsen.
* 🧪 **Fehler auslesen:** JavaScript-Konsole kompakt als Alert ausgeben.

## 🚀 Installation & Nutzung
1. Erstelle einen neuen Apple-Kurzbefehl auf deinem iPad.
2. Füge die Aktion **"JavaScript auf Webseite ausführen"** hinzu.
3. Kopiere den Code aus der `script.js` in das Feld.
4. Aktiviere **"Im Teilen-Blatt anzeigen"**.
5. Öffne eine Website in Safari, tippe auf **Teilen** und wähle **WebInspector 27**.
